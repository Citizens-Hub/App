## CCU 船只与 Melt Value 披露

凡船只或载具未标注为 “Original Concept” 或 “OC”，即表示其可能通过折扣 Warbond 升级组装而成。

因此：

- 其 melt value（RSI 信用兑换价值）将低于当前零售价。
- 这不会影响其升级价值。
- 升级价值始终对应 RSI 商店中的完整零售价。

我们不会因 melt value 低于零售价而批准退款申请。

CCU 船只和载具旨在供持有或进一步升级之用，而非兑换为 RSI 积分。任何熔毁或兑换此类物品的决定，均由买家自行承担风险。
