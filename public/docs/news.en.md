# Patch note 1.4.0

This version delivers a full refactor of the automatic route planner and officially renames it to **“Explore”**

The core goal is to help you more clearly explore feasible Warbond / price-increase CCU paths within a specified time range, with the ability to apply rule constraints and perform interactive filtering directly on the results page.

## Highlights of This Update

* Build a path graph within a specified time range, generating candidate paths based on historical available prices and SKU validity periods.
* Quickly exclude specific CCUs or SKUs and instantly recalculate results.
* Improved interaction linkage: when hovering over a SKU, the corresponding price history curve is highlighted on the right-side chart for easy comparison.
