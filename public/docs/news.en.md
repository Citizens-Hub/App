# Bug Fix Notice: Save Path Failure Issue

A bug was recently discovered that could cause the **save path operation to fail**.
This issue was unintentionally introduced during the recent update involving changes to **WB history data**.

## Details

Due to adjustments made to the WB history data handling logic, a side-effect occurred where the application might fail to correctly save the path under certain conditions.
This resulted in some users being unable to persist their selected save path.

## Status

**The issue has been fully resolved.**
A fix has been implemented and deployed, ensuring that save paths now work as expected.

We sincerely apologize for any inconvenience this issue may have caused.
Thank you for your patience and understanding, and we appreciate your continued support.
