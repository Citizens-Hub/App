# 星际公民订阅商店展示项目

![image](https://github.com/user-attachments/assets/f8043f8a-4838-4f94-93b7-bd9cc2ff9495)

## 技术栈

- React 19
- TypeScript
- Material UI v7
- TailwindCSS v4
- Vite
- ESLint
- React Joyride

## 功能特性

- 响应式设计,支持桌面和移动端显示
- 商品图片轮播展示
- 购物车功能
- 价格汇率转换
- 分页展示
- 本地化新手引导功能
  - 自定义中文界面组件
  - 自定义动画引导指示器
  - 首次访问自动展示
  - 可通过右下角问号按钮手动触发
  - 逐步介绍界面主要功能
